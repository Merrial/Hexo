容器相关

  • docker run

  • docker ps <容器名>
  • docker logs <容器名>

Docker run的常用你选项

**docker run [OPTIONS] IMAGE [COMMAND] [ARG… ] **

选项说明

  • -d 后台运行容器
  • -e 设置环境变量
  • –expose /-p 宿主端口：容器端口
  • –name 指定容器名称
  • –link 连接不同容器
  • -v 宿主目录：容器目录，挂载磁盘卷
